QUICK START
================================================================================

1. Install Dependencies:
   cd python
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

2. Configure:
   cp .env.example .env
   nano .env  # Set your credentials

3. Generate SSL:
   cd ssl && ./generate_cert.sh && cd ..

4. Test:
   python test_connection.py

5. Start:
   ./start.sh

6. Access:
   https://localhost:8501

CONFIGURATION
================================================================================

Required Environment Variables:
- DASHBOARD_PASSWORD      - Login password
- DB_HOST, DB_PORT        - Database connection
- DB_SERVICE, DB_USER     - Database credentials
- DB_PASSWORD             - Database password
- ORDS_BASE_URL           - ORDS endpoint
- ORDS_USERNAME           - ORDS username
- ORDS_PASSWORD           - ORDS password

Optional Settings:
- SSL_ENABLED             - Enable SSL (default: true)
- SESSION_TIMEOUT_MINUTES - Session timeout (default: 30)
- MAX_LOGIN_ATTEMPTS      - Login attempts (default: 3)
- LOG_LEVEL               - Logging (default: INFO)

  - Restart application

  - Check database performance
